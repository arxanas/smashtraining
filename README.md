# Smash Training

_Smash Training_ is a spaced-repetition trainer for Super Smash Bros Ultimate. You can check it out at https://ssb.fit.

# Project setup

Set up the development environment:

```
npm install
```

Compile and run for development:

```
npm run serve
```

Run tests:

```
npm run test
```
